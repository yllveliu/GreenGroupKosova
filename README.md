# Green Group Kosova Website

Professional B2B website for Green Group Kosova, an agricultural company focused on mushroom production and supply.

---

## 🚀 Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- Framer Motion
- Lucide Icons

---

## 🌐 Features

- Multilingual support (Albanian / English)
- Responsive design (mobile, tablet, desktop)
- Product showcase:
  - Fresh Mushrooms
  - Canned Mushrooms
  - Flower Substrate
- Business-focused UI (B2B oriented)
- Contact integration (Viber)

---

## 📦 Installation

```bash
npm install
npm run dev
